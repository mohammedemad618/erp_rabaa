# Travel ERP Role-Based Training Guide

## 1) Training Goals
1. Ensure every role can execute its core workflow without guidance.
2. Reduce pilot support tickets caused by role confusion.
3. Standardize policy, approval, and finance behavior.

## 2) Role Curriculum Matrix
| Role | Core Skills | Target Session |
|---|---|---|
| Employee | Create request, submit expenses, follow policy findings | 60 min |
| Manager | Approve/reject with notes, handle exceptions | 45 min |
| Travel Desk | Move requests through review and booking, update booking records | 60 min |
| Finance | Review expenses, run finance sync, close trip | 75 min |
| Admin | Policy versioning, activation scheduling, audit export | 75 min |
| Auditor | Read-only validation and audit trail export | 30 min |

## 3) Session Structure (Per Role)
1. Business process overview (10 min)
2. Live walkthrough in pilot environment (15-30 min)
3. Hands-on lab (15-25 min)
4. Q&A and known pitfalls (10 min)

## 4) Hands-On Labs

### Lab A: Employee
1. Create a new request.
2. Submit request and confirm policy level.
3. Submit one expense claim with receipt metadata.

### Lab B: Manager + Travel Desk
1. Approve manager step.
2. Complete travel desk review.
3. Confirm booking and save booking details.

### Lab C: Finance
1. Approve submitted expense.
2. Execute finance sync.
3. Validate closure readiness.
4. Execute `close_trip`.

### Lab D: Admin
1. Create policy draft.
2. Simulate policy against a scenario.
3. Activate version immediately.
4. Export audit CSV.

## 5) Competency Checklist
- User can complete role-specific workflow with no trainer intervention.
- User can explain at least two common failure reasons and recovery steps.
- User can identify where to report issues and what evidence is required.

## 6) Assessment Format
- Practical completion checklist
- Time-to-complete measurement
- Error count per participant

## 7) Training Artifacts to Keep
- Attendance list
- Role-level completion scores
- Frequently asked questions
- Open gaps requiring product or documentation updates
